The Enumerable#map Method

What it does and how it works

4-11-2015

The Enumerable module provides a set of really useful methods that allow you to traverse, manipulate, search through, and sort data collections such as arrays and hashes. In this post, we're gonna talk about the Enumerable#map method. The map method, also known as collect, allows you to transform each element in an array without actually modifying the original array itself. How does it do this? Simple: it just creates a transformed copy without altering the original.

For example:
array = [1, 2, 3, 4, 5] creates a simple array
plus_one = array.map { |i| i + 1 } creates a copy of array with each value incremented by 1

In the example above, array is not modified. We just created a copy and changed the copy to what we wanted. What really makes this cool is the fact that we can then join another method to it! For example:
plus_one.delete_at(2) returns 4 in IRB
Now, if we print plus_one in IRB, we get => [2, 3, 5, 6]
And if we print array in IRB, we get => [1, 2, 3, 4, 5]
See how that works? .map accepts a block of code that subsequently acts on each element in the collection collection that it is invoked on but returns the modified collection while leaving the original untouched.
